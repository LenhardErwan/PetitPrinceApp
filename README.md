# Petit Prince Application
Projet d'application mobile dans le cadre de nos études avec le framework [Ionic](https://ionicframework.com).<br/>
<br/>
Développement d'une application pour smartphone à destination des parents d'élèves de l'école maternelle «Le petit Prince».

## Utilisation

Pour visualiser le projet sur un navigateur, effectuez les commandes suivantes :
```
npm install
ionic build
ionic serve
```

Pour compiler le projet sur **android**, effectuez les commandes suivantes :
```
npm install
ionic build
npx cap sync android
npx cap open android
```
Une fois sur Android Studio, c'est à vous de jouer.
